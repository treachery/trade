# BE-020 日频特征工厂

- **类型**：后端/算法
- **优先级**：P2
- **状态**：待办

---

## 1. 需求目标

把原始 OHLCV 转换为统一特征快照，供信号和相似检索使用。

## 2. 需求范围

- 收益、波动、趋势、量能、形态、市场状态特征
- 按 symbol/date 存储
- 特征版本化

## 3. 依赖关系

- `BE-012`
- `BE-002`

## 6. 数据结构示例

```json
{
  "symbol":"600519",
  "date":"2026-06-29",
  "return_20d":0.083,
  "volatility_60d":0.018,
  "ma_20":1500.2,
  "rsi_14":63.1,
  "market_regime":"RANGE"
}
```

## 7. 验收标准

- [ ] 任意 symbol/date 可查询特征
- [ ] 前 N 日不足返回 null
- [ ] 特征带 feature_version
