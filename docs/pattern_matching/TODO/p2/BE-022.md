# BE-022 信号扫描 API

- **类型**：后端/API
- **优先级**：P2
- **状态**：待办

---

## 1. 需求目标

提供批量信号扫描入口。

## 2. 需求范围

- 按交易池扫描
- 支持指定 as_of_date
- 返回成功/失败/无信号列表
- 打印后端请求日志

## 3. 依赖关系

- `BE-021`
- `BE-010`

## 5. API / 函数设计

```http
POST /api/pattern/signals/scan
Content-Type: application/json

{
  "symbols": ["600519", "000858"],
  "as_of_date": "2026-06-29",
  "signal_types": ["buy", "sell"]
}
```

## 6. 数据结构示例

```json
{
  "ok": true,
  "signals": [],
  "failed": [{"symbol":"000001","reason":"数据不足"}],
  "diagnostics": {"elapsed_ms": 3120, "scanned": 2}
}
```

## 7. 验收标准

- [ ] 批量扫描不因单只失败中断
- [ ] 返回 diagnostics
- [ ] 日志包含入参摘要和耗时
