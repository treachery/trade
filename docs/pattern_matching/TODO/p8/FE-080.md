# FE-080 策略治理看板

- **类型**：前端
- **优先级**：P8
- **状态**：待办

---

## 1. 需求目标

展示信号和策略长期表现、漂移和治理建议。

## 2. 页面 / 组件范围

- 策略版本列表
- 后验收益图
- 信号胜率漂移图
- 降权/下线建议

## 3. 按钮与交互

- **筛选**：按策略版本/信号类型/时间筛选
- **查看版本**：打开策略参数快照
- **标记下线**：人工标记策略下线
- **导出报告**：导出治理报告

## 4. 展示数据

- `strategy_version`：策略版本
- `expected_vs_realized`：预期 vs 实际
- `signal_win_rate`：滚动胜率
- `drift_alert`：漂移告警

- `BE-082`
