# FE-050 Walk-Forward 配置页

- **类型**：前端
- **优先级**：P5
- **状态**：待办

---

## 1. 需求目标

用户配置并启动 Walk-Forward 样本外验证。

## 2. 页面 / 组件范围

- 运行配置页
- 参数校验区
- 运行进度区

## 3. 按钮与交互

- **启动验证**：提交配置并返回 run_id
- **保存模板**：保存常用参数
- **停止运行**：取消正在运行的任务
- **查看报告**：跳转 FE-051

## 4. 展示数据

- `simulation_start/end`：验证区间
- `trading/reference universe`：交易池和参考池
- `max_position_pct`：单标的最大仓位
- `cost settings`：交易成本参数

## 5. 依赖关系

- `BE-050`
