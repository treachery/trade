# FE-051 Walk-Forward 报告页

- **类型**：前端
- **优先级**：P5
- **状态**：待办

---

## 1. 需求目标

展示样本外验证绩效和逐日决策依据。

## 2. 页面 / 组件范围

- 绩效总览
- 净值曲线
- 回撤曲线
- 逐笔交易表
- 每日决策日志

## 3. 按钮与交互

- **切换基准**：切换显示基准曲线
- **查看交易详情**：展开信号/相似片段/策略评分
- **导出报告**：导出CSV/HTML
- **回放某日**：查看某个模拟日的完整输入输出

## 4. 展示数据

- `portfolio_value`：组合净值
- `benchmark_value`：基准净值
- `trade_details`：逐笔交易
- `decision_logs`：每日决策
- `stats`：收益/回撤/胜率/换手率

## 5. 依赖关系

- `BE-052`
