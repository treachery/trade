# FE-010 股票池配置组件

- **类型**：前端
- **优先级**：P1
- **状态**：待办

---

## 1. 需求目标

让用户能配置交易候选池和研究参考池。

## 2. 页面 / 组件范围

- 股票池配置页/弹窗
- 交易池选择区
- 参考池选择区
- 自定义代码输入区

## 3. 按钮与交互

- **保存配置**：保存当前交易池和参考池
- **测试加载**：调用后端检查股票数量和无效代码
- **清空**：清空自定义代码
- **导入CSV**：批量导入代码和名称

## 4. 展示数据

- `trading_universe.symbols`：允许买卖的股票
- `reference_universe.symbols`：用于相似检索的股票
- `invalid_symbols`：无法识别或无数据的股票

## 5. 依赖关系

- `BE-010`

## 6. 示例图 / 页面草图

```text
+--------------------------------------------------+
| 股票池配置                                        |
| [交易池]  ( ) A500  ( ) 自定义  [导入CSV]          |
| 代码输入: 600519,000858,...                       |
|                                                  |
| [参考池]  (x) A500  ( ) SP500  ( ) 港股大市值      |
|                                                  |
| [测试加载] [保存配置] [清空]                       |
| 结果: 交易池10只 / 参考池500只 / 无效2只           |
+--------------------------------------------------+
```

## 7. 接口调用

`GET /api/universe/options`、`POST /api/universe`

## 8. 页面状态

- **加载中**：显示 skeleton 和数量统计
- **校验失败**：展示无效代码列表
- **保存成功**：返回 universe_id

## 9. 验收标准

- [ ] 明确区分交易池和参考池
- [ ] 必须展示股票数量
- [ ] 无效代码不能静默忽略
