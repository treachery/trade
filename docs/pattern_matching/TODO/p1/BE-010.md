# BE-010 股票池管理

- **类型**：后端/API
- **优先级**：P1
- **状态**：已完成 ✅

---

## 1. 需求目标

管理交易候选池和研究参考池，明确两者用途边界。

## 2. 需求范围

- 支持 A500/SP500/港股大市值/自定义股票池
- 保存 universe 配置
- 交易池只允许实际买卖，参考池只用于相似检索和统计

## 3. 依赖关系

- `BE-001`

## 4. 示例图 / 流程图

```mermaid
flowchart LR
  UI[股票池配置] --> API[/api/universe]
  API --> T[trading_universe]
  API --> R[reference_universe]
  T --> Signal[信号扫描/买卖]
  R --> Retrieval[相似检索/胜率统计]
```

## 5. API / 函数设计

```http
GET /api/universe/options
POST /api/universe
GET /api/universe/{universe_id}
```

## 6. 数据结构示例

```json
{
  "universe_id": "ref_a500_20260629",
  "universe_type": "reference",
  "symbols": [{"symbol":"600519","name":"贵州茅台","market":"cn"}]
}
```

## 7. 验收标准

- [x] API 返回股票代码和名称
- [x] 交易池/参考池类型不可混淆
- [x] 自定义列表可保存和读取
