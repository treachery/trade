# FE-060 持仓再评估页面

- **类型**：前端
- **优先级**：P6
- **状态**：待办

---

## 1. 需求目标

展示当前持仓每日再评估结果和动作建议。

## 2. 页面 / 组件范围

- 持仓评分列表
- 漂移详情图
- 新机会对比
- 决策时间线

## 3. 按钮与交互

- **刷新评估**：拉取最新持仓评分
- **查看漂移**：展示实际走势 vs 参考片段
- **查看替代机会**：对比新机会TopN
- **导出决策**：导出每日建议

## 4. 展示数据

- `hold_score`：持仓性价比
- `pattern_drift_score`：形态漂移
- `expected_return`：后续预期收益
- `risk_action`：风控建议
- `decision_type`：最终动作
