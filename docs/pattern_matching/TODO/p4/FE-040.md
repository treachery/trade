# FE-040 策略评估排名面板

- **类型**：前端
- **优先级**：P4
- **状态**：已完成 ✅

---

## 1. 需求目标

展示各候选策略在相似情境下的表现和风险。

## 2. 页面 / 组件范围

- 最佳策略卡片
- 策略排名表
- 收益分布图
- 风险提示区

## 3. 按钮与交互

- **按稳健评分排序**：默认排序方式
- **查看收益样本**：展开每个策略样本收益
- **切换指标**：切换中位数/胜率/回撤排序
- **导出**：导出策略评估CSV

## 4. 展示数据

- `strategy_id/name`：策略ID和名称
- `sample_count`：样本数
- `median_return/win_rate`：核心收益指标
- `worst_quantile_20/max_drawdown`：尾部风险
- `bootstrap_lower/upper`：置信区间
- `confidence`：置信度
