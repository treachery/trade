### 2.2 关键字段示例

```python
@dataclass
class SignalEvent:
    event_id: str               # "sig_{symbol}_{date}_{type}"
    symbol: str                 # "600519"
    signal_date: str            # "2026-06-29"
    signal_type: str            # "breakout_20d_high"
    signal_category: str = ""   # "buy" / "sell" / "risk_off"
    direction: str = ""         # "buy" / "sell"
    strength: float = 0.0       # 0.0~1.0
    strength_label: str = ""    # "极强"/"强"/"中"/"弱"
    valid_next_trade_date: Optional[str] = None  # T+1
    decay_rate: float = 0.0     # 衰减率
    is_active: bool = True      # 信号是否有效

@dataclass
class StrategyEvaluation:
    strategy_id: str
    sample_count: int = 0
    median_return: float = 0.0
    win_rate: float = 0.0
    worst_quantile_20: float = 0.0
    max_drawdown: float = 0.0
    sharpe_ratio: float = 0.0
    calmar_ratio: float = 0.0
    bootstrap_lower: Optional[float] = None
    bootstrap_upper: Optional[float] = None
    robust_score: float = 0.0
    confidence: str = ""        # HIGH / MED / LOW
    returns: List[float]        # 样本收益列表（供前端画图）
```

## 3. 序列化接口

所有类统一提供 `to_dict()` 方法，递归处理嵌套 dataclass/list/dict：

```python
def to_dict(self) -> dict:
    return _to_dict(self)  # 内部递归转换
```

## 4. 时序逻辑

```
API 请求 → Flask 接收 JSON → 业务逻辑创建对象 → to_dict() → jsonify 返回
```
