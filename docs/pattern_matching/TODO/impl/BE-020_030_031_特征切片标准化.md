## 2. 数据结构

### 2.1 特征向量（19维）

```json
{
  "t20": [r5d, r10d, r20d, vol_zscore, breakout_strength, rsi_14],
  "t40": [drawdown20d, ma20_slope, ma60_slope, vol20d, range_ratio],
  "t60": [trend120d, trend60d, vol_pctile, dist_52w_high, dist_52w_low],
  "env": [ma60_distance, vol_pctile, breadth_20d]
}
```

### 2.2 特征维度说明

| 维度 | 名称 | 计算方式 | 范围 |
|------|------|----------|------|
| t20[0] | r5d | close / close[-5] - 1 | 收益率 |
| t20[1] | r10d | close / close[-10] - 1 | 收益率 |
| t20[2] | r20d | close / close[-20] - 1 | 收益率 |
| t20[3] | vol_zscore | (vol - mean_20) / std_20 | z-score |
| t20[4] | breakout_strength | close / max(high[-20:]) - 1 | -1~0 |
| t20[5] | rsi_14 | RSI(14) / 100 | 0~1 |
| t40[0] | drawdown_20d | close / max(high[-19:i+1]) - 1 | 回撤 |
| t40[1] | ma20_slope | (ma20[i] - ma20[i-5]) / ma20[i-5] | 斜率 |
| t40[2] | ma60_slope | (ma60[i] - ma60[i-5]) / ma60[i-5] | 斜率 |
| t40[3] | vol_20d | 滚动20日收益率标准差 | 波动率 |
| t40[4] | range_ratio | (high20 - low20) / low20 | 振幅 |
| t60[0] | trend_120d | close / close[-120] - 1 | 趋势 |
| t60[1] | trend_60d | close / close[-60] - 1 | 趋势 |
| t60[2] | vol_pctile | 当前波动率在历史分位 (0~1) | 分位 |
| t60[3] | dist_52w_high | close / max(high[-250:i+1]) - 1 | 距高点 |
| t60[4] | dist_52w_low | close / min(low[-250:i+1]) - 1 | 距低点 |
| env[0] | ma60_dist | (close - ma60) / ma60 | -1~1 |
| env[1] | vol_pctile | 同上 t60[2] | 0~1 |
| env[2] | breadth | 近20日上涨天数/20 | 0~1 |

## 4. 标准化流程

```
原始特征提取 → 收集所有窗口的原始特征 → 按列计算 mean/std
→ 对每个窗口执行 (value - mean) / std → features_norm
→ 查询标的用同样的 mean/std 执行标准化
```
