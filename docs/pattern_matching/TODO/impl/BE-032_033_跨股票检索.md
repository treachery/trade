# BE-032 跨股票检索 + BE-033 检索API — 实现文档

## 1. 模块位置

`pattern_matching/retrieval.py` + `app.py` 路由

## 2. 核心算法：四级递进检索

```mermaid
flowchart TD
    Q[查询标的 60日特征] --> NORM[用参考库 statistics 做 z-score]
    NORM --> L1["L1: t-20 近期形态<br/>加权欧氏距离 × 全库<br/>→ Top100"]
    L1 --> L2["L2: t-40 中期结构<br/>在 Top100 内检索<br/>→ Top50"]
    L2 --> L3["L3: t-60 长期背景<br/>在 Top50 内检索<br/>→ Top10"]
    L3 --> L4["L4: 市场环境过滤<br/>在 Top10 内重新排序"]
    L4 --> R["最终 Top10<br/>(含股票代码/名称/时间段/K线/前瞻)"]
```

## 4. 内部函数接口

### 4.1 主入口

```python
def analyze_symbol(symbol: str,
                   as_of_date: str = None,
                   adjust: str = "qfq",
                   lookback_years: int = 6,
                   top_k: int = 10,
                   reference_symbols: list = None,
                   ref_limit: int = 50) -> dict:
```

**流程：**
1. 信号侦测 query stock
2. 加载 query stock 完整历史
3. 加载 reference pool 各股票历史
4. 跨股票构建片段库（build_multi_stock_windows）
5. 提取 query 标的特征
6. 提取 query 标的近120日K线
7. 执行多股票递进检索（multi_stock_retrieval）
