## 3. 接口

### 3.1 主函数

```python

                   as_of_date: str = None,     # 分析日期; None=最新

    """
    Point-in-Time: 仅使用 as_of_date 及以前数据
    返回: {ok, symbol, as_of_date, buy_signals[], sell_signals[], ...}
    """
```

### 3.2 内部指标计算

| 函数 | 用途 |
|------|------|
| `_rsi(closes, period=14)` | RSI 指标 |
| `_bollinger(closes, period=20, num_std=2)` | 布林带 (upper, middle, lower) |
| `_volatility(closes, period)` | 滚动波动率 |
| `_market_regime(closes, ma_period=60)` | BULL/BEAR/RANGE 分类 |

## 4. 时序逻辑

```mermaid
sequenceDiagram
    participant Caller
    participant detect_signals
    participant load_kline
    participant engine

    Caller->>detect_signals: detect_signals(symbol, as_of_date)
    detect_signals->>load_kline: load_kline(symbol, start, end)
    load_kline-->>detect_signals: DataFrame
    detect_signals->>detect_signals: Point-in-Time filter (date <= as_of_date)
    detect_signals->>engine: _sma/_rsi/_macd/_atr/_bollinger
    engine-->>detect_signals: indicator arrays
    loop for each signal rule
        detect_signals->>detect_signals: 检测信号条件
        alt 触发
            detect_signals->>detect_signals: 创建 SignalEvent
        end
    end
    detect_signals-->>Caller: {buy_signals, sell_signals, features_snapshot}
```

## 5. 信号确认原则

- T 日收盘确认信号
- T+1 日可执行交易
- 信号有有效期：入场窗口 N 日
- 超过有效期进入衰减期
- 完全过期后信号失效
