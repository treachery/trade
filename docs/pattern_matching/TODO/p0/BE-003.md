# BE-003 A股交易约束实现

- **类型**：后端/交易模拟
- **优先级**：P0
- **状态**：待办

---

## 1. 需求目标

实现 Walk-Forward 和组合模拟所需的 A 股交易约束。

## 2. 需求范围

- T+1
- 整数手
- 涨跌停
- 停牌
- 佣金/印花税/滑点

## 3. 依赖关系

- `BE-001`
- `BE-002`

## 4. 示例图 / 流程图

```mermaid
flowchart TD
  order[Order] --> suspended{停牌?}
  suspended --是--> skip1[跳过:停牌]
  suspended --否--> limit{涨跌停?}
  limit --不可成交--> skip2[跳过:涨跌停]
  limit --可成交--> tplus{T+1?}
  tplus --违反--> skip3[跳过:T+1]
  tplus --通过--> lot[按100股取整]
  lot --> cost[计算成本]
  cost --> done[成交]
```

## 7. 验收标准

- [ ] 涨停无法买入
- [ ] 跌停无法卖出
- [ ] 当日买入不可当日卖出
- [ ] 买入股数按 100 股取整
