# FE-031 递进检索链路面板

- **类型**：前端
- **优先级**：P3
- **状态**：待办

---

## 1. 需求目标

解释 Top10 相似片段是如何逐层筛出来的。

## 2. 页面 / 组件范围

- 检索链路折叠面板
- 阶段候选表
- 特征权重展示

## 3. 按钮与交互

- **展开/收起**：展开某一层候选列表
- **复制 query_id**：复制检索日志ID
- **查看权重**：展示该层使用的特征和权重

## 4. 展示数据

- `stage_logs.t20.input/output`：t20层输入/输出数量
- `candidate_scores`：候选片段得分
- `feature_weights`：该层特征权重
- `retrieval_config_version`：检索配置版本

## 5. 依赖关系

- `BE-033`
