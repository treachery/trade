- **类型**：QA
- **优先级**：P3
- **状态**：待办

## 1. 需求目标

防止出现空K线、没有股票代码、只检索自身历史、点击无反应等问题。

## 2. 需求范围

- API字段完整性测试
- 跨股票结果测试
- 前端按钮点击测试
- K线非空测试
- 页面日志测试

## 3. 依赖关系

- `BE-033`
- `FE-030`

## 5. API / 函数设计

```python
assert result["similar_windows"][0]["symbol"]
assert len(result["query_kline"]["ohlc"]) > 0
assert len(result["similar_windows"][0]["kline_before"]) > 0
assert len(set(w["symbol"] for w in result["similar_windows"])) >= 2
```

## 7. 验收标准

- [ ] API 至少返回 5 个相似片段
- [ ] 至少包含 2 个不同 symbol
- [ ] 每个片段有前半段和后半段K线
- [ ] 页面点击开始分析后有可见日志
