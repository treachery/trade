# FE-040 策略评估排名面板 — 实现文档

## 1. 组件结构

```
EvalPanel
├── BestStrategyCard (最佳策略突出展示)
│   ├── 策略名称 (大号)
│   ├── 置信度标签 (彩色)
│   ├── 中位数收益 / 胜率 / 尾部风险 / Bootstrap CI
│   └── 风险提示 (样本不足时)
├── EvalTable (可排序排名表)
│   ├── 表头可点击 (中位数/胜率/回撤/评分)
│   └── 每行含折叠的 returns 样本列表
├── BoxplotCanvas (Canvas箱线图, 前8个策略)
└── ExportButton (导出CSV)
```

## 2. 交互逻辑

- 默认按 robust_score 降序
- 点击表头切换排序字段和方向
- 展开某行显示 returns 样本列表
- 点击导出下载 CSV
- Canvas箱线图: min - Q1 - median - Q3 - max

## 3. Canvas箱线图数据

每个策略的 returns 数组排序后:
- min = returns[0]
- Q1 = returns[25%]
- median = returns[50%]
- Q3 = returns[75%]
- max = returns[-1]
