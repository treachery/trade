# 股票策略回测平台

一个轻量、自用的 A 股策略回测平台：拉取历史日线 → 应用策略 → 在 K 线图上标注买卖点 → 给出总回报等绩效。内置**「连续阴线反转」**策略。

## 快速开始

```bash
# 1. 安装依赖（已为你装好；如换环境）
pip install -r requirements.txt

# 2. 启动
python app.py        # 或双击 run.bat
# 浏览器打开 http://127.0.0.1:5000
```

## 内置策略：连续阴线反转

- **买入**：当日之前出现「连续阴线根数 **>** N」（默认 N=5，即超过连续 5 根阴线 ≥6 根），
  且当日**由阴转阳**（收盘>开盘）并且当日**涨幅 > 1%**。
- **卖出**（满足任一即卖）：
  1. 收盘价**跌破买入价**（止损）；
  2. 出现**连续 3 根阴线**。
- 成交：以信号当日**收盘价**撮合；A 股按 100 股/手取整；可设手续费率。
- 区间结束仍持仓则按最后一日收盘强制平仓以计算总回报。

> 阴线 = 收盘<开盘；阳线 = 收盘>开盘。所有阈值均可在界面调整。

## 绩效指标
策略总回报、年化收益、同期买入持有回报（基准）、期末资金、交易次数、胜率、单笔平均收益、最大回撤；并提供 K 线买卖点图与资金曲线图、逐笔交易明细。

## 数据来源
[akshare](https://akshare.akfuncs.com)（免费、无需 token，数据来自东方财富），首次拉取某股票会联网，之后读本地缓存。

## 开源股票/量化策略平台调研对比

| 平台 | 语言 | 数据 | 接入方式 | 策略支持 | 历史回测 | 适合 |
|---|---|---|---|---|---|---|
| **Backtrader** | Python | 自带接口少，需自接(yfinance/akshare/CSV) | 库 import | 事件驱动，指标/多品种/多周期完善 | 强，支持滑点/手续费/再投资 | 个人/研究，最流行通用回测 |
| **Backtesting.py** | Python | 自接 | 库 import | 轻量，单品种为主 | 强，内置可视化(bokeh) | 快速验证单策略（与本项目最接近） |
| **VectorBT** | Python | 自接 | 库 import | 向量化，超快、可大规模参数寻优 | 极快，适合网格搜索 | 批量回测/因子研究 |
| **Zipline(-reloaded)** | Python | bundle 机制(美股为主) | 库 import | 事件驱动，Quantopian 同源 | 强 | 学术/美股 |
| **vn.py** | Python | 实盘券商/期货接口丰富 | 框架+GUI | 全栈：回测+实盘+CTA | 有 | 国内实盘交易，期货/股票 |
| **RQAlpha(米筐)** | Python | A股内置较好 | 框架 | A股股票/期货 | 强 | A股策略研究 |
| **QUANTAXIS** | Python | A股全市场自带 | 框架 | 数据+回测+模拟 | 有 | A股数据中台 |
| **Qlib(微软)** | Python | A股/美股，含数据集 | 框架 | 机器学习量化为主 | 有 | AI/因子选股 |
| **QuantConnect/LEAN** | C#/Python | 云端海量(全球) | 云平台/本地 | 全栈 | 强 | 多资产、云端 |

**选型结论（针对你的需求：自用、A股、单策略回测、看买卖点与回报）**：
- 最契合的是 **Backtesting.py / Backtrader** 这类轻量库 + **akshare** 数据。
- 本项目即按此思路自建：**akshare(数据) + 自研逐日引擎(完全可控的策略逻辑) + Flask/ECharts(可视化买卖点)**，比直接用框架更易精确实现你描述的规则并定制界面。
